## Getting Started

### Prerequisites

- Ruby >= 3.0.2
- Node >= v18.19.1
- Yarn >= 1.22.22

### Installation

1. Clone the repo
```sh
git clone https://github.com/Malloti/fullstack-todo.git
```
2. Install the dependencies and setup
```sh
bin/setup
```
3. Run the application
```sh
bin/dev
```
